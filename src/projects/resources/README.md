# 学习资源

这里整理了经过我们实际使用验证的开发资源、学习材料和实用参考。

## 资源类型

### 官方文档
- 重要技术栈的官方文档精读笔记
- 版本更新要点与迁移指南
- 官方最佳实践总结

### 教程指南
- 分步式学习教程与路径建议
- 常见问题解决方案集合
- 实战项目完整实现过程

### 工具集合
- 开发效率工具推荐与配置
- 设计资源与素材库
- 开发环境配置清单

## 筛选标准

所有收录资源均符合：
- ✅ **亲自验证**：团队成员实际使用过
- ✅ **保持更新**：定期检查资源有效性
- ✅ **质量优先**：内容准确、结构清晰
- ✅ **免费可及**：尽可能选择开源或免费资源

## 使用建议

1. **按需查找**：根据当前学习或项目需求针对性查阅
2. **实践结合**：资源配合实际编码效果最佳
3. **贡献反馈**：发现更好资源或有更新建议欢迎补充
